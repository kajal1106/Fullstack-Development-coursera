RWD Media Queries

Media query is a CSS technique introduced in CSS3. The @media rule is used in media queries to apply different styles for different media types/devices. Media queries can also be used to change layout of a page depending on the orientation of the browser. It uses the @media rule to include a block of CSS properties only if a certain condition is true.
